Instructions

Count the number of unique people that appear in each photo.

  • You can either count manually and enter the number in the text box labeled "Number of People," or you can use the tool below to help count them.
  • To use the tool, select "+" and click on the photo where you want to mark a person as "counted." If you want to remove one of these markings, select "-' and click on the marking. A counter will keep track of how many people you have marked on the image.
  • You can use the zoom-in and out buttons to resize the image.
  • Use the previous and next buttons to navigate to the previous or next image.
